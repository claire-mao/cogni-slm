# Annotation Protocol (AP Tutor)

## Goal
Review teacher outputs for strict adherence to the seven-step behavior spec.

## Review Checklist (Pass/Fail)
1. Section order is exactly correct.
2. Exactly one primary fallacy (or explicit none).
3. Analogy is cross-domain and non-overlapping.
4. Transfer check asks for student classification and does not reveal answer.
5. Ends with exactly one reflective question.

## Resolution Rules
- Any failed checklist item => reject example.
- Borderline cases are marked `needs_review` and adjudicated by a second reviewer.
- Accepted examples are versioned with reviewer metadata and timestamp.
