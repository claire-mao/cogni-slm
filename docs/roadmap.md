# Cogni One-Week Roadmap

## Day 1
- Lock problem, behavior spec, PRD, and architecture.
- Verify baseline model inference path.

## Day 2
- Finalize eval harness and prompt-test baseline.
- Build labeling batch and AP teacher prompt/schema.

## Day 3
- Generate/filter/dedup/split real dataset.
- Run first training in Colab.
- Produce first base-vs-tuned numbers.

## Day 4
- Perform error analysis.
- Improve data for identified failure modes.
- Retrain and re-evaluate.

## Day 5
- Final report package, dataset artifact, model artifact, and demo.

## Milestone Gates
- M1: Behavior and eval contract locked.
- M2: Dataset pipeline reproducible and versioned.
- M3: First successful tuned run with artifacts.
- M4: Base-vs-tuned evaluation published.
- M5: Iteration evidence documented.
