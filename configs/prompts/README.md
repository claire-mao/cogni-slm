# Prompt Versioning Store

This directory stores production teacher prompt versions and metadata managed by:

- `src/teacher/prompt_versioning.py`

Managed artifacts:

- `registry.json`
- `events.jsonl`
- `prompts/<prompt_id>/<version_id>/prompt.txt`
- `prompts/<prompt_id>/<version_id>/metadata.json`
- `changelogs/<prompt_id>.md`
