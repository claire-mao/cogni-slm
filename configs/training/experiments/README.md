# QLoRA Experiment Configs

This directory contains the full QLoRA hyperparameter experiment matrix design.

Files:

- `qlora_matrix_v1.json`: complete search space definition and full-factorial counts.
- `qlora_phase1_ofat_v1.json`: one-factor-at-a-time screening runs.
- `qlora_phase2_interactions_v1.json`: interaction-focused matrix blocks.
- `qlora_phase3_full_factorial_v1.json`: exhaustive full-factorial execution plan.
- `manifest.json`: registry of matrix files and run-count summaries.
